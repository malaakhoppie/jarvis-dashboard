# Backtest

> Mr. Casino reflection task backtesting session. Review charts, track progress, give feedback.

## Session Start

Load the current task criteria and greet Mala. Ask which task she is working on if not specified:

---

### Task 1 — Liquidity Marking (4hr)
**Goal:** 100 reps total. 30 minimum to start.
**What to mark on each chart:**
- Pure doji within a previous wick at a non-FU high or low
- Big wicks (notable)
- HCS (High Confluence Structure)

**Rules:**
- 4hr timeframe only
- No entry decisions — marking only
- Each chart = roughly 1 month of price action
- Do 10x3 (10 charts, 3 markup passes) = 30 screenshots

**What to send:** Screenshot of marked 4hr chart. One chart per message or batch them.

---

### Task 2 — Direction & Top Down (4hr → 15m → 1m)
**Goal:** 30 examples minimum.
**Process per example:**
1. Start on 4hr — identify liquidity from Task 1
2. Drop to 15m — which liq is holding with HCS + negation? Which is being swept?
3. Drop to 1m — confirm LTF manipulation building in that direction
4. Label each: LOW liq move or HIGH liq move. State which side wins and why.

**Rules:**
- Pick your best Task 1 examples as starting points
- Must show all 3 timeframes per example
- 10 minimum, 30 recommended

---

### Task 3 — Full System + Hard Rules

**3.1 — Environment audit:**
- Write out your current trading setup, session times, and daily process
- Identify and remove any self-sabotage factors before touching charts

**3.2 — Add 11hr or Daily TF:**
- Add the 11hr (or daily if no access) on top of Task 2 setups
- Mark HTF POI — zone + aligned TFS + major liquidity
- This gives the bigger picture direction context

**3.3 — Entry filter:**
- Hard rule: **No 10min HCS + negation = no trade**
- Run 10+ examples on replay/sim
- HTF POI → session timing → news → 10min entry
- Log every instance: valid entry taken, invalid entry skipped

---

## How to Review Screenshots

When Mala sends a screenshot, check against the current task criteria:

**Task 1 review checklist:**
- [ ] Is it on the 4hr?
- [ ] Are dojis within a previous wick at a non-FU high/low?
- [ ] Are big wicks marked?
- [ ] Are HCS marked?
- [ ] Any missed liquidity?
- [ ] Any incorrectly marked (FU candle marked as doji, wrong side, etc)?

**Task 2 review checklist:**
- [ ] Does the 4hr show clear liquidity targets?
- [ ] Does the 15m show which side has HCS + negation holding?
- [ ] Does the 1m confirm LTF manipulation building?
- [ ] Is the direction call correct — low liq vs high liq clearly labelled?

**Task 3 review checklist:**
- [ ] Does HTF POI align with direction?
- [ ] Is session timing accounted for?
- [ ] Does a valid 10min HCS + negation exist?
- [ ] Would this be a valid entry by the rules?

## Progress Tracking

After each session, state:
- Current task
- Rep count for the session
- Total reps completed (ask Mala to confirm running total)
- What was sharp
- What needs correction
- What to focus on next session

---

## Feedback Style

- Direct and specific — point to exact candles or zones in the screenshot
- No padding — if it's wrong say what's wrong and why
- If it's right, confirm and move on
- Reference exported Discord examples or methodology rules where relevant
